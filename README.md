# Studio60 Challenge V2

Full-stack bilingual Studio60 Challenge V2 submission.

Live frontend:

```txt
https://studio60-challenge-v2.netlify.app
```

GitHub repository:

```txt
https://github.com/housss77/studio60-challenge-v2
```

## Stack

- Laravel 13
- Statamic 6 CMS
- Gatsby 5
- React 18

## Structure

- `backend-statamic`: Laravel + Statamic backend and CMS content model
- `frontend-gatsby`: Gatsby static frontend deployed to Netlify

## Features

- Bilingual French / English
- Statamic-managed homepage content
- Statamic-managed services
- Statamic-managed projects
- Statamic CMS management
- Custom Laravel JSON API endpoints for content export
- Static Gatsby build for Netlify deployment
- Responsive landing page
